Este sistema apresenta uma proposta simples para organizar
emprestimos de uma biblioteca.

Para iniciar os empréstimos é necessário cadastrar o usuario gerente
com o usuario 'gerente@gerente.com' a senha fica a critério do usuario.

O usuario pode emprestar os livros disponiveis, para prazos de 7, 14 e 21 dias.

A devolução é controlada pelo gerente.

Para o usuario há a pagina inicial que exibe separadamente os itens emprestados
de acordo com seu status de emprestimos, mais de 7 dias em dia, menos de 7 dias para
devolver, e vencidos.

Páginas para atualizar seus dados.

Pagina para emprestar os livros disponíveis.


Para o gerente há a pagina de atualizar seus dados, controle de emprestimo,
onde é exibido os livros que estão emprestados e os usuarios que emprestaram
com um botão que permite cancelar o empréstimo.

Uma pagina que exibe um formulário de cadastro dos livros.

E a página inicial que exibe todos os livros cadastrados.
